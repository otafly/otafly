# OTAFLY
## Service Development
### Swift
### Install MySQL
1. brew install mysql
2. brew services restart mysql
3. mysql -uroot
4. create database otafly;
### Setup Project
1. Open project via Package.swift
2. Edit Scheme -> Run -> Options -> Working Directory -> Set to /path/to/your/project
## Web Development
### Install Node
1. brew install node
### Setup Project
1. npm install
2. npm start
